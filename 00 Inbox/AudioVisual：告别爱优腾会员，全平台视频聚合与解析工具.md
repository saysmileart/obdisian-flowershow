## AudioVisual：多平台视频解析与观看工具

AudioVisual 是一款基于 Electron 的开源视频聚合与解析工具，支持腾讯视频、爱奇艺、优酷、哔哩哔哩、芒果TV 等平台。本地运行、隐私友好、自动更新，提供多解析接口与简洁界面，跨 Windows、macOS、Linux 使用。仅供学习交流，严禁商业用途。

## ✨ 工具亮点

### 多平台覆盖

支持腾讯视频、爱奇艺、优酷、哔哩哔哩、芒果TV等常见视频平台，可在一处管理多平台内容，提高观看效率。

内置多种稳定解析接口，提升解析稳定度，帮助使用者更顺畅地完成视频获取与播放。

采用现代化设计风格，界面清晰明快，支持主题切换，适合长时间使用。

### 跨平台支持

基于 Electron 构建，可在 Windows、macOS 与 Linux 上运行，使用环境更灵活。

提供内置更新管理，可检测版本变化并引导用户完成安装，保持使用体验持续稳定。

## 🚀 使用指南

### 系统要求

- Node.js ≥ 16.0.0
- npm ≥ 8.0.0
- 系统支持：Windows 10+、macOS 10.14+、Ubuntu 18.04+

### 安装方式

克隆项目后安装依赖并启动应用：

```bash
git clone https://github.com/RemotePinee/AudioVisual.git
cd AudioVisual
npm install
npm start
```

### 基本操作

- 选择平台与视频内容
- 选择解析接口
- 点击 Parse! 执行解析
- 在美韩日剧模式下，可直接通过剧名搜索播放

### 更新体验

当检测到新版本时，会提示下载更新并显示进度，完成后可选择直接安装。
