## Caesium Image Compressor 是什么

Caesium Image Compressor 是一款以速度与画质著称的图片压缩软件，支持 JPG、PNG、WebP、TIFF 等主流格式，为摄影师、内容创作者、电商卖家与企业提供稳定高效的图像优化体验。软件可在保持出色画质的情况下，将图片压缩至原来的 10%–40%，大幅减少存储空间与传输成本。

软件提供简洁的界面、实时预览、批量处理与自定义输出配置，让不同使用场景都能快速获得理想的压缩效果。对于希望进一步掌控细节的用户，也提供元数据保留、文件夹结构保留等进阶功能。

## 核心亮点

### 高压缩比，画质依然在线

Caesium 提供有损与“接近无损”的多档压缩质量调节：

- 支持自定义压缩质量，按需求在体积与画质之间找到平衡
- 提供无损压缩与尺寸调整功能，既能缩小体积，也能改成统一尺寸
- 通过直观的前后对比预览界面，压缩效果一目了然

对摄影师、设计师、运营同学来说，这种“高压缩比 + 高画质”的组合，适合日常备份、发给客户、上传到网站与社交平台。

### 批量处理，一次性瘦身整套相册

Caesium 专门为批量场景做了优化：

- 支持拖拽整个文件夹，批量导入上千张图片
- 多张图片同时处理，充分利用多核性能
- 可保留原有文件夹结构，导出时不打乱你的素材归档方式
- 输出路径可自定义，避免覆盖原图，方便对比与回滚

无论是清理硬盘、给网站做图床瘦身，还是给公司素材库做统一压缩，都能显著节省时间。

### 多平台支持，桌面 + 命令行 + 在线全覆盖

### 面向开发者的源码构建

Caesium 提供公开源码，开发者可以按需本地构建：

- 使用 Rust 编译核心库 `libcaesium` （依赖 `cargo` ）
- 使用 Qt6 SDK（例如 6.8.0 开源版本）构建桌面界面
- macOS 平台额外依赖 libssh 与 Sparkle（支持自动更新）
- 通过 CMake 配置各平台构建参数，按需指定 Qt 与相关依赖路径

想要追求自定义功能或二次开发时，可以从 GitHub 仓库获取源码，选择标记的稳定版本进行编译。
