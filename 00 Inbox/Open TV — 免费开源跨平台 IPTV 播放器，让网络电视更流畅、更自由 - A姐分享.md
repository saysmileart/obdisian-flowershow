### 🧩 技术依赖与安装说明

Open TV 使用 mpv、ffmpeg 和 yt-dlp 作为核心依赖。

- macOS 用户可通过 Brew 或 MacPorts 安装
- Fedora 用户需启用 rpmfusion
- Debian、Ubuntu 用户推荐使用 backport 以获取最新 yt-dlp
- Windows 版本自带 mpv，无需额外安装

命令示例：

```bash
brew install mpv ffmpeg yt-dlp  # macOS  
sudo apt install mpv ffmpeg yt-dlp  # Debian/Ubuntu
```
