## GeekAI概览

GeekAI 是一套基于大语言模型 API 构建的开源 AI 助手解决方案，拥有完整的前后端系统，安装后即可投入使用。平台整合了 OpenAI、Claude、通义千问、Kimi、DeepSeek、Gitee AI Serverless 等主流模型，并提供丰富的 AI 角色应用、插件能力与可视化运营管理后台，适合个人创作者、企业团队与开发者快速搭建属于自己的 AI 服务。

## 核心优势

- **完整一体** ：前端应用 + 后台运营管理，安装即用。
- **多模聚合** ：支持 OpenAI、Claude、通义千问、Kimi、DeepSeek，兼容 Gitee AI Serverless。
- **顺滑交互** ：WebSocket 带来打字机式输出体验。
- **角色即服务** ：内置小红书写手、英语翻译大师、苏格拉底、孔子、乔布斯、周报助手等角色应用。
- **AI 绘画能力** ：集成 MidJourney / Stable Diffusion，支持文生图、图生图、换脸、融图等流程。
- **音乐生成** ：兼容 Suno 文生音乐。
- **插件生态** ：提供插件 API，可结合 function 调用扩展；内置微博热搜、今日头条、今日早报与 AI 绘画函数插件。
- **支付与会员** ：集成支付宝、微信支付；支持会员套餐与点卡。亦可使用个人微信二维码作为充值渠道。
- **PLUS 增强** ：更友好的 UI、DALL·E 文生图、文生思维导图、为模型/角色绑定专属 API Key、站点 Logo 与版权信息配置。
